AluminumOS

Windows, built for everyone.

What is it?

AluminumOS offers custom Windows builds that are free from adware and come pre-installed with essential software tailored to the specific edition you select.

Edition Explanation

All editions except Paranoid come with Firefox pre-installed.

Workspace Edition

The Workspace Edition is designed to cater to developers while also serving as a versatile solution for general workspace needs. It provides a robust environment tailored to development tasks, along with a range of tools and features suitable for diverse professional applications.

  • VSCode + Visual Studio
  • Latest Python + Pip
  • Java + JDK
  • Audacity
  • FFMPEG

Community Edition

The Community Edition includes open-source and free alternatives to popular software, such as LibreOffice, and is specifically tailored for casual users. It offers a range of essential tools and applications designed to meet everyday computing needs.

  • LibreOffice
  • WSL

Paranoid Edition

The Paranoid Edition is equipped with free antivirus software and the Tor Browser pre-installed, providing enhanced security and privacy features for users with heightened concerns about online safety.

  • Avast Antivirus (Free Edition)
  • Tor Project Browser

Basic ISO

The Basic ISO provides a clean, ad-free Windows environment with no pre-installed software, offering a straightforward foundation for users to customize and build their system according to their individual needs.
